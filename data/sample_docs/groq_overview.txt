Groq is an AI inference company that provides extremely fast LLM inference using custom LPU (Language Processing Unit) hardware. Groq API supports models like LLaMA3-70B and Mixtral-8x7B. Groq achieves speeds of 500+ tokens per second making it ideal for real-time agentic applications. The API is compatible with OpenAI SDK format. Groq offers a free tier with generous rate limits suitable for development and prototyping.
