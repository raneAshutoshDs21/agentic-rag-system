Agentic AI systems are AI architectures where LLMs act as autonomous agents that can reason, plan, use tools, and execute multi-step tasks. Key components include a router that directs queries, a reasoning agent that plans steps, tools like web search and code execution, memory for context persistence, and an evaluation agent for quality control. Agentic systems use ReAct, Chain-of-Thought, and Tree-of-Thought for complex reasoning tasks.
